Commuter Alarm Clock

Description

Commuter Alarm helps you solve the problem of arriving late for work due to unpredictable traffic conditions. It works by regularly checking the traffic while you are sleeping to adjust your wakeup time so that you are never late to wherever you need be.

Through the use of the Google Maps APIs, it can access real time traffic conditions to give you a reliable ETA to your destination.

Features

  • Allows for multiple alarms, checking traffic simultaneously.
  • Uses Google's Maps APIs for accurate and reliable traffic conditions.
  • Easily enter your destination with autocompletion using Google's Places APIs.
  • Supports older versions of Android.

Food War

Description

FoodWar is a simple tower defense game written in Objective-C and developed using SpriteBuilder, a game developing suite that builds iOS and Android applications.

Choose from four condiments to keep from the evil picnic food from reaching their goal.

Features

  • Built using the SpriteBuilder game development suit using Objective-C.
  • Available on iOS and Android devices.
  • Choose from multiple towers and fight against array of different enemies.
  • Features 10 levels with upgradable towers.
